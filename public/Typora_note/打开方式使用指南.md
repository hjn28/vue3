## 根据新的编码方式打开新的tab页

```js

操作指南：（直接调用方法）
window.top. openXtgnToIframeTab(xtbm,obj);
// xtbm 介绍（Xtbm:新电局业务系统需要传递yyid，金三系统传递zyy_dm;）
# xtbm两种传参方式：
1、调用税务人应用查询接口 queryTaxerAppInfoList()
2、没有税务人应用查询接口

# obj格式如下；
const obj ={
    id:dbsyGndm, //【自定义，不可相同，具有惟一性，最好定义为与业务相关的】
    title : '首页', //【新打开页面的主题】
    tabContentType : 'iframe',
    src :dbsyurl, //【打开新页面的url】
    isCloseBtn : 'false', //【true:页面显示关闭按钮。false:页面无关闭按钮】
    tabItemWidth : 150,
    isShowMenuItem : [ "false", "true", "true", "true" ],				
    tempData:gnmc //【自定义参数，在门户右上角显示当前的操作页面，如果为空，则默认显示title。】
};
```

### xtbm两种传参方式(有接口/无接口)：

```js
# 分为有接口的查询和无接口的；
// 有接口的情况下 queryTaxerAppInfoList()
	1、xtbm可传空串，业务自己根据上述接口返回的域名+地址自行拼接地址，拼接完成后放入obj.src中
    2、xtbm可传上述接口中返回的appId【应用ID】---【门户根据appId自己发请求获取域名】，此时obj.src中只传入地址即可，无需拼接域名，方法里面会进行地址拼接的处理，此外方法中可兼容src前是否携带斜杠的情况，可放心传入参数。
	// 返回的数据如下：
        [
          {
             "appId": "10000003665",------【应用ID】
             "appName": "应用名称",
             "appShortName": "应用简称",
             "domainName": "域名"--------【域名】
           },
           {
             "appId": "20000003665",
             "appName": "应用名称2",
             "appShortName": "应用简称2",
             "domainName": "域名2"
            }
        ]
// 无接口的情况下
	1、xtbm可传字符串'XDZSWJN1'，obj.src中只传入地址即可，无需拼接域名，方法里面会进行地址拼接的处理，此外方法中可兼容src前是否携带斜杠的情况，可放心传入参数。
```
